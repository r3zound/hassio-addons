## 关于

FlexGet 是一个多用途的媒体自动化工具。请访问 https://flexget.com/

默认密码是 'homeassistant123'，或可以在插件选项中设置

## 需求

## 配置

    "PUID": 自定义用户 (默认 root)
    "PGID": 自定义用户 (默认 root)
    "WebuiPass": webui 密码定义
    "FG_PLUGINS": 插件添加
    "FG_LOG_LEVEL": 日志级别 (critical|error|warning|info|verbose|debug|trace)

## 启动

- 启动插件。等待一段时间并检查日志以查看是否有错误。
