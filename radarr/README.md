## 关于

[Radarr] 是一个用于 Usenet 和 BitTorrent 用户的电影收藏管理器。它可以监视多个 RSS 源以获取新电影，并将与客户端和索引器接口，以获取、排序和重命名电影。它还可以配置为在库中更好的质量格式可用时，自动升级现有文件的质量。
