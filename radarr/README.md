## 关于

[Radarr] 是一个供 Usenet 和 BitTorrent 用户使用的电影收藏管理器。它可以监控多个 RSS 源以获取新电影，并与客户端和索引器接口以抓取、排序和重命名它们。它还可以配置为在库中存在的文件的更好质量格式可用时，自动升级这些文件的质量。
