官方MariaDB附加组件的web界面。

## 关于

phpMyAdmin是一款MySQL和MariaDB的数据库管理工具。常用操作（管理数据库、表、列、关系、索引、用户、权限等）可以通过用户界面执行，同时您仍然可以直接执行任何SQL语句。

此附加组件专为管理官方Home Assistant MariaDB附加组件而设计。
