## 关于

[Prowlarr] 是一个索引器管理器/代理，基于流行的 *arr .net/reactjs
基础堆栈构建，用于与您的各种 PVR 应用程序集成。 Prowlarr 支持对
Torrent Trackers 和 Usenet Indexers 的管理。它与
Lidarr、Mylar3、Radarr、Readarr 和 Sonarr 无缝集成，提供对
您的索引器的完整管理，无需为每个应用设置索引器（它自动处理所有）。
